- [Empowering Netflix Engineers with Incident Management | Molly Struve via Netflix TechBlog (September 2025)](https://netflixtechblog.com/empowering-netflix-engineers-with-incident-management-ebb967871de4) - Netflix used to have one central team responsible for managing incidents. This didn't scale as the organisation grew and several learning opportunities were missed. Netflix has since adopted [incident.io](https://incident.io/) to enable teams to manage their own incidents
- [Importing a frontend Javascript library without a build system | jvns.ca (November 2024)](https://jvns.ca/blog/2024/11/18/how-to-import-a-javascript-library/) - We can use vanilla JS or ES modules in your frontend using script tags and [import maps](https://developer.mozilla.org/en-US/docs/Web/HTML/Reference/Elements/script/type/importmap). This article covers how you can identify what sort of files you're working with, how to import these files, and approaches for handling CommonJS libraries or ES Modules with several dependencies
